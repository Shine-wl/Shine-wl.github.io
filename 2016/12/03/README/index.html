搭建总结

本地的操作：

  • 安装 node 环境
  • 安装 git
  • 安装 hexo
    • npm install --global hexo-cli
  • 在本地生成博客
    • hexo init 博客目录名称
  • 在本地预览
    • hexo server

准备码云线上仓库：

  • 注册码云账号
    • 注意：需要单独验证邮箱
  • 创建一个仓库
    • 仓库的名字随便起
  • 在仓库页面找到 服务 -> Pages
    • 选择启动服务
  • 拿到仓库地址
    • https://git.oschina.net/lpzdemayun/lpzdemayun.git
