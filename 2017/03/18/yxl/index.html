原型链

对象属性的搜索访问机制：原型链

  • 当你访问对象的某个成员的时候：
    1. 先在自己身上找，找到即返回使用
    2. 找不到，则沿着原型链向上查找
    3. 如果在上一个原型对象中找到，则返回使用
    4. 如果找不到，则继续沿着原型链查找
    5. 直到 Object.prototype
    6. 如果还找不到，则继续找Object.prototype.proto
    7. 这个时候发现 Object.prototype.proto 是 null
    8. 则停止查找，返回 undefined
  • 原型链的末端是谁？null
  • 原型链倒数第二个是谁？Object.prototype
