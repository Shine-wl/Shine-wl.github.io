call方法 和 apply方法 以及 bind方法

call方法和apply方法

  • 函数的call方法可以实现改变this指向 然后调用该函数
  • 相同点：
  1. apply 和 call 的作用一样
  2. 第一个参数也是用来指定内部的 this 指向的
  3. fn.apply(this)和fn.call(this)
  4. call 和 apply 的返回值就是被调用函数的返回值

  • 不同点：apply 和 call 的区别就在于传参的方式不一样

    • call方法
      1. 可以传多个参数

    • apply方法

      1. apply 必须传数组
        1. apply 会把参数数组给展开，然后一一对应的传递给 fn 方法
        2. apply 只处理第一个参数参数，其它的不处理

bind方法

  • bind方法也是用来改变函数内部this指向的，但是他不调用，二食返回一个指定了this环境的新函数；
  • 注意：这个方法只是指定this的指向，但是不会执行，只是返回一个指定this指向的新函数
  • javascript中的传参方式是先求值之后在进行传参
  • bind的参数的问题
  1. bind 可以传参，但是不调用
  2. 得到bing之后的新函数也可以进行传参
  3. 当你在调用通过 bind 得到的新函数的时候，际上会把你在 bind ,的时候传递的参数和你调用新函数传递的参数进行合并，然后作为函数的参数
